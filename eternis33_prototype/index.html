ETERNIS-33: PROTOTYPE SHARD

Discover the Prisms. Uncover the truth.

Project Overview

Eternis-33: Prototype Shard is an AR mobile game set in a cyberpunk world where players collect glowing crystal "Prisms" anchored to real-world GPS coordinates.

UI Preview

Spectra Mood Ring Demo

Twilight palette, dynamic emotional halo, optional camera overlay and ambient hum.

Core Features

AR Visualization

Experience the cyberpunk world of Eternis-33 through your device's camera as you discover glowing Prism crystals in the real world.

GPS Integration

Prisms are anchored to real-world locations, encouraging exploration and discovery in your environment.

Collection Mechanics

Tap Prisms to collect them and add to your inventory, building your collection of these mysterious data crystals.

Multiplayer Persistence

Prisms are visible to all players in the same location, creating a shared experience and encouraging social interaction.

Cyberpunk Aesthetic

Immerse yourself in a neon-drenched visual style with glowing effects and a futuristic interface.

Rich Lore

Uncover the mysteries of Eternis-33 as you collect Prisms and learn about this data-infused world.

Technical Implementation

Engine & Platforms

  • Engine: Godot 4.x
  • Platforms: Android (ARCore), iOS (ARKit)
  • AR Plugins: godot-arcore (Android), equivalent for iOS
  • Map/GPS: OpenStreetMap + Mapbox API
  • Networking: WebSocket / lightweight backend
